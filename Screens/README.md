**Farm Signals** is a comprehensive agricultural management and monitoring platform designed to provide farmers, agricultural officers, and stakeholders with real-time insights into crop health, field intelligence, and resource management.

---

## 🌟 Key Features

### 📡 Monitoring & Analytics
- **NDVI Monitoring**: Satellite-based Normalized Difference Vegetation Index tracking to monitor crop health.
- **Field Intelligence**: Geographic insights and geospatial data analysis for precise farming.
- **Analytics Dashboard**: Interactive charts and data visualizations (NDVI trends, crop health distribution, scout status, survey completion).
- **Scouting & Reports**: Track field issues, damage reports, and schedule overdue scouts efficiently.

### 👥 Farmer & Field Management
- **Farmer Management**: Comprehensive records for farmers and users.
- **Villages & Sections**: Organize operations geographically.
- **Varieties & Plots**: Detailed tracking of crop varieties and individual land plots, including geo-tagging.
- **Officers**: Manage field officers and their assigned territories.

### 📋 Operations
- **Survey Management**: Create, assign, and track field surveys for continuous data collection.
- **Responsive Design**: Accessible from both desktop and mobile devices seamlessly.

## 🚀 Getting Started

To explore the user interface locally:

1. Clone or download the repository to your local machine.
2. Navigate to the `Screens` directory.
3. Open `signin.html` in any modern web browser to start the user flow. (Use any dummy credentials to log in and proceed to the dashboard).

### File Structure
- `Screens/`: Contains all HTML frontend views including:
  - `dashboard.html`: The main overview and metrics page.
  - `signin.html`, `signup.html`: Authentication views.
  - `ndvi_monitoring.html`, `field_intelligence.html`: Advanced agricultural modules.
  - `plots.html`, `users.html`, `surveys.html`: Management interfaces.
  - `images/`: Static graphic assets and backgrounds.

## 🛠️ Technology Stack

- **HTML5 & CSS3**: Semantic structure with modern styling (Variables, Grid, Flexbox).
- **Vanilla JavaScript**: DOM manipulation and interactivity.
- **Chart.js**: Dynamic data visualization on the dashboard and analytics pages.
- **FontAwesome**: Scalable vector icons for the UI.
- **Google Fonts**: Utilizing the 'Outfit' font family for a modern aesthetic.
